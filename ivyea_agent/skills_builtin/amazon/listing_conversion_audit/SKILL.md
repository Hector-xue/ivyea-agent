# Amazon Listing Conversion Audit

Use this skill when ad data suggests a conversion problem or when the user asks
to improve listing content, images, A+ content, bullets, title, or keyword fit.

## Workflow

1. Collect listing text, product promise, price point, review context if
   available, and the winning or wasting search terms.
2. Map search terms to title, bullets, image claims, A+ modules, and expected
   customer intent.
3. Mark gaps as traffic relevance, message mismatch, proof weakness, offer
   weakness, or image clarity issue.
4. Produce concrete listing tasks, not generic copy advice.
5. When image or A+ files exist locally, inspect the available text/assets before
   making recommendations.

## Output

Return: conversion diagnosis, missing intent coverage, content fixes, image/A+
tasks, ad actions that should wait for listing changes, and expected follow-up
metric to watch.
