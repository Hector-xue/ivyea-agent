# Amazon Weekly Account Review

Use this skill for weekly or periodic account review, campaign health summaries,
and next-action planning.

## Workflow

1. Start with target ACoS, account/store/ASIN scope, time window, and data source.
2. Summarize account health: spend, sales, ACoS, winners, waste, no-order spend,
   budget constraints, and listing feedback.
3. Compare current actions against previous memory and rejected decisions.
4. Produce a prioritized action queue and separate non-ad listing tasks.
5. End with a clear next review date and metrics to check.

## Output

Use this structure: executive summary, KPI movement, winners, waste, budget/bid
actions, listing feedback, blocked/risky actions, queued actions, next week plan.
