# Amazon Launch Playbook

Use this skill for launch-stage ASINs, cold-start ad structure, early keyword
discovery, and deciding when to move from exploration to efficiency.

## Workflow

1. Confirm launch stage, budget ceiling, margin, initial core keywords, and
   protected discovery terms.
2. Keep automatic and broad discovery useful, but isolate obviously irrelevant
   traffic.
3. Harvest early winners conservatively into controlled manual campaigns.
4. Avoid suppressing relevant but low-conversion discovery terms before listing,
   review, price, and image readiness are checked.
5. Define the next review window and minimum data threshold.

## Output

Return a 7/14/30-day action plan with discovery structure, harvest criteria,
negative keyword criteria, budget pacing, listing readiness checks, and stop-loss
conditions.
