Windows / macOS / Linux pipx 一键安装 v1.1.5 · 离线包可内网部署
# 1 · 一键安装（自动装 pipx，拉 Release wheel）
$ curl -fsSL https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.sh | bash
# 2 · 选模型 / 配 key（可选接领星）
$ ivyea onboard
# 3 · 开始对话
$ ivyea chat
→ 对话式运营 Agent 就绪 ✓
# 1 · 一键安装（PowerShell）
PS> iwr https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.ps1 -UseBasicParsing | iex
# 2 · 选模型 / 配 key
PS> ivyea onboard
# 3 · 开始对话
PS> ivyea chat
→ 对话式运营 Agent 就绪 ✓
# 1 · 用 pipx 从源码安装
$ pipx install git+https://github.com/Hector-xue/ivyea-agent.git
# 2 · 配置主脑模型与密钥
$ ivyea config
# 3 · 开始对话
$ ivyea chat
→ http://127.0.0.1:8765 （ivyea serve 给 IvyeaOps 用）

Why Ivyea Agent

为什么要单独做一个 Amazon Agent

通用编码 Agent 解决不了否词误伤、预算节奏和真实写入审批；这些运营问题，确定性 + 可控才靠谱。

◆

规则引擎 + LLM 复核

决策由确定性规则给出，LLM 只做词分类、归因与措辞，不推翻数据。

⊘

审批式写入

否词 / 调 bid / 预算逐条人工确认，operate 总开关默认关，幅度闸 + 审计回滚。

◐

影子模式

动钱前先只记不写，用后续真实花费回测「若照做能省 / 能多卖多少」。

⌂

本地私有

配置、API key、记忆、知识库默认都在本机 ~/.ivyea/，不依赖外部数据库。

⟐

多模型可切

DeepSeek / Claude / Gemini / Codex / Qwen / Kimi / Ollama 等 provider，带重试与降级链。

▤

记忆 + 知识库

SQLite FTS5 长期记忆 + 持久指令注入；内置 Amazon 运营知识与本地统一检索。

Capabilities

一个 CLI，运营 + 工程双内核

◉

对话式 Agent

更多：周度复盘 · 竞品诊断 · Listing 转化诊断 · 成本护栏 · 动作队列 · 运行 trace
01

读数据

领星 OpenAPI 拉店铺广告，或读搜索词报告 CSV。

02

出候选

规则引擎按否词/收割/调bid/预算分区，每条带证据与置信度。

03

LLM 复核

词分类、归因、护栏与措辞复核，不改动数据结论。

04

审批写入

逐条确认后写回，抓快照、记审计，可一键回滚。

也能嵌入 IvyeaOps 工作台
