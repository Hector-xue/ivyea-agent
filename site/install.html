Install guide · v1.0.38

安装 Ivyea Agent

推荐使用 pipx 安装。它会把 ivyea 作为独立命令放进 PATH，不污染系统 Python，也方便升级和卸载。

最快安装

Linux / macOS
curl -fsSL https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.sh | bash
Windows PowerShell
iwr https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.ps1 -UseBasicParsing | iex

安装完成后重开终端，执行：

ivyea onboard
ivyea chat

系统要求

项目要求
PythonPython 3.9 或更高版本。
系统Windows、macOS、Linux（CI 三平台 × py3.9/3.11/3.12 全绿）。
安装器脚本自动安装 pipx；也可手动安装。
模型可使用 DeepSeek、OpenAI 兼容模型、Anthropic、Gemini 等配置。

固定版本安装

团队内建议固定版本，避免不同成员安装到不同 Release。

curl -fsSL https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.sh | IVYEA_VERSION=v1.0.38 bash
$env:IVYEA_VERSION="v1.0.38"
iwr https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.ps1 -UseBasicParsing | iex

手动安装

pipx install git+https://github.com/Hector-xue/ivyea-agent.git

开发者可用 editable 模式：

git clone https://github.com/Hector-xue/ivyea-agent.git
cd ivyea-agent
python -m venv .venv
. .venv/bin/activate
pip install -e ".[dev,anthropic]"

离线部署

在有外网的发版机上生成离线包（含 wheelhouse 与启动器），复制到内网后解压安装，全程不下载 Python 包：

python scripts/build_offline_bundle.py
# 产物在 dist/offline/：
#   ivyea-agent-offline-1.0.19.tar.gz   (Linux / macOS)
#   ivyea-agent-offline-1.0.19.zip      (Windows)
# 内网解压后：
bash install.sh            # Linux / macOS
# 或 powershell -ExecutionPolicy Bypass -File .\install.ps1   (Windows)

要把本地语义检索依赖或 embedding 模型也放进离线包，加 --with-semantic（详见 README）。

首次配置

ivyea onboard      # 引导：选模型 → 配 key → 可选领星 → AGENTS.md
ivyea model list
ivyea doctor

配置和记忆默认在 ~/.ivyea/，包括 .env、settings.json、mcp.json、memory.db 和用户知识库。

故障排查

问题处理方式
找不到 ivyea运行 pipx ensurepath 后重开终端。
Python 版本太低安装 Python 3.9+；Windows 安装时勾选 Add Python to PATH。
Release 读取失败私有仓库设置 GITHUB_TOKEN，或设置 IVYEA_REF=main 从 git 安装。
中文 CSV 乱码优先使用 .xlsx，或把 CSV 另存为 UTF-8。
