安装 Ivyea Agent

推荐使用 pipx 安装。它会把 `ivyea` 作为独立 CLI 放进 PATH，不污染系统 Python。

快速安装

Linux / macOS

curl -fsSL https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.sh | bash

Windows PowerShell

iwr https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.ps1 -UseBasicParsing | iex

安装后重开终端，执行：

ivyea config
ivyea chat

Windows

  1. 安装 Python 3.9+，勾选 Add Python to PATH。
  2. 以普通 PowerShell 执行一键安装脚本。
  3. 重开 PowerShell 后运行 `ivyea --version`。
$env:IVYEA_VERSION="v0.5.6"
iwr https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.ps1 -UseBasicParsing | iex

Linux

Debian / Ubuntu 如未安装 Python：

sudo apt update
sudo apt install -y python3 python3-pip python3-venv
curl -fsSL https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.sh | bash

macOS

如果没有 Python，可先用 Homebrew 安装：

brew install python
curl -fsSL https://raw.githubusercontent.com/Hector-xue/ivyea-agent/main/scripts/install.sh | bash

升级卸载

pipx upgrade ivyea-agent
pipx uninstall ivyea-agent

卸载不会删除 `~/.ivyea/`，因此配置、记忆和知识库会保留。

离线部署

从 Release 下载 wheel 和依赖 wheel，复制到内网后安装：

pip download ivyea_agent-0.5.2-py3-none-any.whl -d wheelhouse
pipx install --pip-args="--no-index --find-links ./wheelhouse" ./wheelhouse/ivyea_agent-0.5.2-py3-none-any.whl

故障排查

问题处理
找不到 `ivyea`运行 `pipx ensurepath` 后重开终端。
Release 读取失败私有仓库设置 `GITHUB_TOKEN`，或设置 `IVYEA_REF=main` 从 git 安装。
中文 CSV 乱码优先使用 `.xlsx`，或把 CSV 另存为 UTF-8。
真实写入失败运行 `ivyea doctor` 和 `ivyea mcp validate <名称>`。
